# s3explorer

## configure

* create your configuration coffeescript and modify it.  
  `cp src/coffee/aws_init.coffee.sample src/coffee/aws_init.coffee`

## setup

1. install nodejs
2. `. ./run.sh setup`

## develop

1. build jade, coffeescript and sass
  `. ./run.sh build`

1. clean built files
  `. ./run.sh clean`

1. run electron
  `. ./run.sh`
